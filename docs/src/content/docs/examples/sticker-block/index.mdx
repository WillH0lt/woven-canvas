## What This Demonstrates

- **Custom Component**: `StickerData` stores which emoji to display
- **Block Definition**: Enables rotation and proportional scaling
- **Custom Tool**: Toolbar button with smiley face icon
- **Floating Menu Button**: Emoji picker that appears when stickers are selected
