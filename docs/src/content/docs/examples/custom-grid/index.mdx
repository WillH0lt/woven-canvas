## How it works

Configure grid via editor options and update at runtime:

```vue
<WovenCanvas
  :editor="{
    grid: {
      enabled: true,
      colWidth: 40,
      rowHeight: 40,
      strict: false,
    }
  }"
  :background="{ type: 'grid', size: 40 }"
/>
```

Update grid settings at runtime:

```typescript
editor.nextTick((ctx) => {
  const grid = Grid.write(ctx)
  grid.enabled = true
  grid.colWidth = 60
  grid.rowHeight = 60
})
```

## Grid options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `enabled` | `boolean` | `false` | Enable snap-to-grid |
| `colWidth` | `number` | `20` | Grid column width in pixels |
| `rowHeight` | `number` | `20` | Grid row height in pixels |
| `strict` | `boolean` | `false` | Force sizes to grid multiples |
| `snapAngleRad` | `number` | `π/36` | Rotation snap angle (5°) |
| `shiftSnapAngleRad` | `number` | `π/12` | Rotation snap with Shift (15°) |

## Background options

The `background` prop controls the visual grid:

```typescript
background: {
  type: 'grid',    // 'grid' | 'dots' | 'none'
  size: 40,        // Grid cell size
  color: 'rgba(255,255,255,0.1)'  // Line/dot color
}
```

## Key APIs

- `Grid.write(ctx)` - Modify grid settings
- `Grid.snapPosition(ctx, pos)` - Snap a position to grid
- `Grid.snapSize(ctx, size)` - Snap a size to grid
