## How it works

Disable the canvas controls plugin to prevent pan and zoom:

```typescript
const pluginOptions = {
  controls: false,
}
```

Users can still select and manipulate shapes, but cannot move the viewport.

## Alternative: Lock zoom only

To allow panning but lock the zoom level, set equal min/max values:

```vue
<WovenCanvas
  :plugin-options="{
    controls: { minZoom: 1, maxZoom: 1 }
  }"
/>
```

## Key APIs

- `pluginOptions.controls` - Configure or disable the canvas controls plugin
- `Camera.write(ctx)` - Set initial camera position programmatically

## Use cases

- **Slideshows** - Present content at fixed positions
- **Embedded widgets** - Keep content visible in constrained spaces
- **Guided tutorials** - Control what users see during walkthroughs
