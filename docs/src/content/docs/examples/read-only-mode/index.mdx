## How it works

Disable the selection, eraser, pen, and arrows plugins to prevent editing:

```typescript
const pluginOptions = {
  selection: false,
  eraser: false,
  pen: false,
  arrows: false,
}
```

The canvas controls plugin remains enabled so users can still pan and zoom.

## Key APIs

- `pluginOptions` prop on `WovenCanvas` - Configure or disable built-in plugins
- `:key` prop - Force remount when changing plugin configuration

## Use cases

- **Presentations** - Display canvas content without allowing modifications
- **Embedding** - Show canvas content in documentation or portfolios
- **Permissions** - Toggle based on user authentication/authorization
